# Telco Customer Churn Prediction

## Overview

This project predicts customer churn for a telecommunications company using machine learning models.

The goal is to identify customers who are likely to leave the company, so that the business can target them with retention actions.

## Dataset

The dataset used in this project is the Telco Customer Churn dataset from Kaggle.

It contains customer demographic information, account details, subscribed services, contract type, payment method, and churn status.

## Project Steps

- Data cleaning
- Exploratory Data Analysis
- Feature encoding
- Train/test split
- Feature scaling
- Class imbalance handling
- Model training
- Model evaluation
- Business interpretation

## Models Used

- Logistic Regression
- Neural Network
- XGBoost

## Evaluation Metrics

The models were evaluated using:

- Accuracy
- Precision
- Recall
- F1-score
- ROC-AUC
- PR-AUC
